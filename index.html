1. Classical Cryptography


This section covers early methods used to hide information. These are simple compared to modern cryptography but are historically important and help learn core concepts.


Caesar Cipher

One of the oldest cipher systems, used by Julius Caesar.


It shifts each letter of the alphabet by a fixed number (classically 3) to turn plaintext into ciphertext.


Example: A → D, B → E, etc.


This type of cipher is a substitution cipher, meaning each original character is replaced with another. Though simple, it introduces core ideas like encryption, decryption, and keys — all essential to more advanced systems.


General Shift Cipher


Like Caesar, but the shift value k can be any number, not just 3.


You choose the key (the shift amount), and that key controls how text is transformed.


Affine Cipher


A slightly more complex substitution cipher.


It uses two keys — one to multiply and one to shift — using the formula:


E(x) = (a * x + b) mod 26


where “a” and “b” are keys and “x” is the numeric letter value.


Transposition Cipher


Instead of replacing letters with other letters (like substitution ciphers), it rearranges the letters.


It moves the positions of characters but keeps the letters themselves unchanged.


Transposition ciphers teach an important concept: encryption doesn’t always change characters — it can change their order.


Caesar Cipher

The Caesar cipher shifts each letter by a fixed number of positions in the alphabet.

2. Modern Cryptography: RSA


The page includes a section on RSA cryptography, a real modern encryption system that’s widely used today.


What RSA Is


Asymmetric encryption: unlike Caesar or affine ciphers, RSA uses different keys for encrypting and decrypting.


Public key = used by anyone to encrypt a message.


Private key = used by the receiver to decrypt it.


RSA is secure because it’s hard to factor very large numbers into primes.


Why RSA Matters


Modern internet security — like HTTPS, VPNs, and secure email — depends on asymmetric systems like RSA. It’s a huge leap from classic ciphers because it protects real-world data.
